# Roadmap

Where Superlog is headed. Each `## ` heading is a column; bullets under it are
items. Plans change as we learn — treat this as direction, not a promise.

## Now

- **AWS resource dashboards** — connect an AWS account, inventory its services, and get a per-service dashboard from metric streams and logs with no agent to run.
- **Service map** — an automatic topology of your services and infrastructure, built from observed telemetry and enriched with context.
- **Anomaly detection** — a proactive agent that sweeps your telemetry and opens incidents before you notice the alert.

## Next

- **Agent-proposed infra fixes** — let an investigation propose a scoped infrastructure change you can approve from the web or Slack.
- **Richer ingest rules** — per-attribute include/exclude rules so you keep the telemetry that matters and drop the noise.
- **More SDKs** — first-class instrumentation for more languages and frameworks.

## Later

- **Deeper cost controls** — spend visibility and budgets per project.
- **More integrations** — issue trackers, incident tools, and chat beyond what ships today.
